Blood Smear Analysis

A website and machine learning application created by Andrew van der Kleut

Purpose:

The purpose of this site is to allow people to upload cell images from a blood smear. The image should contain one cell other than erythrocytes in question for analysis.

The machine learning program was created using TensorFlow with training and validation images from A dataset for microscopic peripheral blood cell images for development of automatic recognition systems
